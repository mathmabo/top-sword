FANTASY RPG (by Franuka) v1.9.6
-------------------------
Thank you for purchasing this asset pack!
It was created from scratch by me.
Feel free to use it on any commercial or non-commercial project (as long as you don't resell the assets), just remember to add a link to my itch page and/or Twitter, it helps a lot :) -->https://twitter.com/franuka_art
---
Included are all the tiles and spritesheets for each asset in three sizes (1x, 2x, 3x), in .png format, transparent background. You should be able to recreate anything you see on the example images.
The tiles are 16x16px (100%), 32x32px (200%) or 48x48px (300%)

CHANGELOG

v1.5: Added metal fences, and 3 crops (carrot, berries and sunflower) all with different grow stages (seed, sprout, fully grown, harvested)
v1.6: Added stone roads, stone tile, a few more flowers and leaves, a rock, and 3 fruits (apple, orange, peach). Also added 200% size versions of each file.
v1.7: Added tileable cliffs, cliff paths and cave entrances. Also added tall iron fences (2 tiles tall, 1 tile wide), and 3x versions of each asset. Added 12 new monsters/characters and colour variations. Thanks Ron for the requests!
v1.8: Added attack, hit and die animations to the Mushy, Rat, Spider, Cultist, Bandit, Yeti, Demon, Orc, Witch and Necromancer sprites. Thanks Jeromy for commissioning these changes.
v1.9: Added a horse sprite, 3 versions (unsaddled, saddled and with rider), Idle and Walk animations. You can combine the "rider" version with any humanoid character sprite. Just put the character in a layer BELOW the horse, at the desired x and y position (see reference for recommended position). Use the character's Idle animation, 1st frame, and adjust the y position to accompany the horse's movement (for best results). Let me know if you have any questions or feedback! :)
v1.9.5: Added waterfall tiles (9 tiles, 5 animation frames each), front view. You can find them in the "Tileset" folder.
v1.9.6: Added forest tiles (dense tree walls). Each of these tiles is made out of 3 normal size tiles (so 48x48px instead of 16x16px, for example). 14 tiles per set, 3 colours.
